往Spring容器初始化bean的方法

2022-06-23 Work

要实现的外部service为例子

1.原始方法(线上环境失效，本地有效)

@Configuration
public class LicenseConfig {
    @Bean
    public LicenseServiceImpl licenseService(){
        return new LicenseServiceImpl();
    }
}
@Autowired
    private LicenseService licenseService = SpringContextHolder.getBean(LicenseServiceImpl.class);

2.改进方法(线上环境有效，本地有效)，使用最原生的方法获取

@Configuration
public class LicenseConfig {
    @Bean
    public LicenseService licenseService(){
        return new LicenseServiceImpl();
    }
}
ApplicationContext context = new AnnotationConfigApplicationContext(LicenseConfig.class);
    LicenseService licenseService = context.getBean("licenseService", LicenseService.class);
