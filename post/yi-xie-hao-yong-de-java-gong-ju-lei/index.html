一些好用的Java工具类

2022-06-28 Works

  • java.util包下的Collections类

    作用场景：适用于排序(sort-reverse)，获取最大值或最小值(max-min)，转换线程安全集合(synchronizedList)，转换不可修改集合(unmodifiablexxx)，返回空集合(emptyList)

  • CollectionUtils工具类

    apache 的 CollectionUtils 工具类比较优先，因支持的方法比较多

    集合判空(isEmpty)，对两个集合进行操作(union-并集，intersection-交集)

  • Objects工具类

    对象判空(isNull)，获取对象HashCode(hashCode)

  • BooleanUtils工具类

    针对Boolean类型进行操作：转换数字toInteger，转换原始的boolean toBoolean

  • org.apache.commons.lang3包下的StringUtils工具类

    优先推荐使用isBlank和isNotBlank方法，因为它会把" "也考虑进去。

    使用 StringUtils 的 split 方法会返回 null

    判断是否是纯数字isNumeric

  • Assert类，它表示断言类

  • org.apache.commons.io包下的IOUtils类 流处理类

    IOUtils 类的write

    IOUtils 类的toByteArray

  • MDC是org.slf4j包下的一个类，它的全称是 Mapped Diagnostic Context，我们可以认为它是一个线程安全的存放诊断日志的容器。

  • 获取对象工具类

    如果你想获取某个对象的所有接口，可以使用 ClassUtils 的getAllInterfaces方法

    如果你想获取某个类的包名，可以使用 ClassUtils 的getPackageName方法

    如果你想判断某个类是否内部类，可以使用 ClassUtils 的isInnerClass方法

  • Http枚举类

    其实org.springframework.http包下的 HttpStatus 枚举，或者org.apache.http包下的HttpStatus接口，已经把常用的 http 返回码给我们定义好了，直接拿来用就可以了，真的不用再重复定义了
